# Configurar

Para empezar a utilizar el Adobe GenStudio con AEM Assets as a Cloud Service y GenStudio for Performance Marketing, es necesario configurar ambas aplicaciones correctamente. El proceso de configuración incluye varios pasos que deben completarse para poder integrar completamente Adobe GenStudio con AEM Assets y GenStudio for Performance Marketing.

Estas tareas requerirán la colaboración de administradores que tengan los permisos adecuados para administrar ambos sistemas.
